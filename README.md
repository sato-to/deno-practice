# deno-practice
- denoのサンプルプログラム

## deno コマンド
```shell
# lint / format
$ deno fmt
$ deno lint

# プログラムの実行
$ deno run main.ts

# コマンド登録
$ deno install --force --name sample-command main.ts

# コマンド解除
$ deno uninstall sample-command
```
